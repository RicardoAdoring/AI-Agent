发现并修复的问题

原前端 SuperAgent.vue 只监听默认 eventSource.onmessage，不会处理 event: step 或 event: final 这类自定义 SSE 事件。因此已调整 HuManus/app/services/manus_app.py，让 Manus SSE 输出默认 data: 文本消息，并在结束前返回日志文件路径，最后发送 [DONE]。

cd old/yu-ai-agent/yu-ai-agent-frontend
npm install
npm run build
npm run dev -- --host 0.0.0.0
