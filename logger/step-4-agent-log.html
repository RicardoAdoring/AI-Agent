HuManus Step 4：YuManus Agent 实现日志

记录日期：2026-05-19

阶段结论：已将 /api/ai/manus/chat 从占位实现升级为安全版 YuManus Agent SSE 工作流，并为每次 Agent 运行生成 HTML 逐步日志。

新增/修改文件

路径说明
HuManus/app/agents/schema.py定义 AgentState、AgentStep、AgentRunResult。
HuManus/app/agents/yumanus.py实现严格 JSON 动作决策、工具调用、最大步数和终止逻辑。
HuManus/app/services/html_step_logger.py实现 Agent 运行 HTML 日志，每个 step 写入磁盘。
HuManus/app/services/manus_app.py封装 YuManusAgent 并提供 SSE 输出。
HuManus/app/api/ai.py替换 Manus 占位接口，新增 /api/ai/manus/tools。

Agent 行为

  • 每次运行最多 MANUS_MAX_STEPS=20 步。
  • LLM 必须输出严格 JSON：thought、action、action_input、final_answer。
  • 每一步只执行一个工具或 final。
  • 工具结果作为 observation 进入下一步上下文。
  • 调用 terminate 或 final 后结束。
  • 运行日志写入 logger/manus-runs/。

SSE 事件

event: step
data: {...}

event: final
data: {...}

data: [DONE]

安全边界

  • 不开放任意终端执行。
  • 只调用 Step 3 注册的安全工具。
  • 文件和 URL 安全校验由工具层统一处理。
  • 日志内容进行 HTML 转义，避免注入。
