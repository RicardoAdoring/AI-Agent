路径说明

默认注册工具

  • read_file：读取 Manus 工作区内 UTF-8 文本文件。
  • write_file：写入 Manus 工作区内 UTF-8 文本文件。
  • web_search：搜索 provider 占位，未配置 provider 时返回明确错误。
  • web_scrape：抓取公开 HTTP/HTTPS 页面，默认阻断内网和本机地址。
  • resource_download：带大小限制下载公开资源到下载目录。
  • generate_pdf：把文本内容生成 PDF 到输出目录。
  • terminate：结束 Agent 循环。

安全边界

重要：任意终端命令执行未开放。TerminalCommandTool 仅作为禁用 stub 存在，不进入默认工具注册表。
  • 文件路径必须在 MANUS_WORKSPACE_DIR、输出目录或下载目录内。
  • 阻止访问 .env、credentials.json、SSH 私钥等敏感文件名。
  • URL 默认阻止 private、loopback、link-local、multicast 地址。
  • 下载资源有最大字节数限制。
  • Web search 未配置 provider 时不伪造结果。
